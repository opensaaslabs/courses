# Registry

The registry is the canonical index for OpenSaaSLabs Academy artifacts and graph relations.

Folders such as `courses/`, `labs/`, and `assessments/` contain source content. The registry contains machine-readable records and relationships that make the academy queryable.

## Registry Goals

- Stable IDs
- Explicit relationships
- Dependency analysis
- Search indexing
- Credential evidence tracking
- Graph export
- API generation
