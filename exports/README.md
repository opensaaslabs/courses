# Exports

Exports are generated machine-readable views of the OpenSaaSLabs Academy source artifacts.

## Generated Outputs

- `catalog/` - normalized catalog collections
- `graph/` - nodes, edges, and full graph
- `jsonld/` - schema.org compatible JSON-LD
- `search/` - search index

Author-facing YAML remains the source. Exported JSON is for websites, APIs, search engines, visualizations, and agents.
