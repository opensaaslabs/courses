# Decision Framework

Major decisions should be captured as AEPs or ADRs.

## Required Questions

- What problem is being solved?
- Who benefits?
- What alternatives were considered?
- What breaks?
- What is the migration path?
- How does this affect conformance?
- How does this affect security and privacy?
- How does this affect portability?
- How will success be measured?
- What quality attributes are improved or put at risk?

## Decision Rule

Prefer specification clarity, interoperability, conformance, and long-term ecosystem health over implementation convenience.
