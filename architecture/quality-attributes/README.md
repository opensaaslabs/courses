# Quality Attributes

Every major proposal, release, schema, contract, implementation, and publishing surface should be evaluated against these quality attributes.

## Attributes

- Interoperability
- Portability
- Reliability
- Auditability
- Traceability
- Maintainability
- Extensibility
- Security
- Privacy
- Accessibility
- Usability
- Performance
- Recoverability
- Observability

## Rule

AEPs and releases should explicitly identify which quality attributes they improve, preserve, or risk.
