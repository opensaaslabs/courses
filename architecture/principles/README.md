# Architecture Principles

## Core Principles

- Everything is a governed artifact.
- Everything has a stable identifier.
- Everything is versioned.
- Everything is reviewable.
- Everything is auditable.
- Everything is traceable.
- Everything is portable.
- Source artifacts are canonical.
- Generated outputs are replaceable views.
- Specification comes before implementation.
- Conformance comes before certification.
- Open standards are preferred over proprietary formats.
- Human accountability is required for publication and credentialing.
