# ADR-0001: Academy as a Learning Graph

## Status

Accepted

## Context

A traditional course repository does not provide enough structure for skills, credentials, evidence, search, analytics, or agentic learning workflows.

## Decision

Model OpenSaaSLabs Academy as a governed learning graph.

## Consequences

- Every artifact requires stable identity.
- Relationships must be explicit.
- Validation is required before publishing.
- Exports become the consumption layer for websites, APIs, analytics, and agents.
