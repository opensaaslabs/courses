# Learning Graph

The OpenSaaSLabs learning graph connects courses, modules, lessons, labs, projects, assessments, skills, competencies, badges, certifications, credentials, and evidence.

## Core Relations

- contains
- teaches
- requires
- awards
- contributesTo
- validates
- evidencedBy
- issuedTo
- backedBy
- mapsTo

## Design Principle

Folders are for authors. Registries are for systems. Graph relations are for reasoning.
