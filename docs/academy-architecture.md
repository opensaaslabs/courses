# OpenSaaSLabs Academy Architecture

OpenSaaSLabs Academy is a governed learning graph built from versioned repository artifacts.

## Layers

1. Content layer: courses, modules, lessons, labs, projects, and assessments.
2. Registry layer: canonical metadata for every artifact.
3. Skills layer: skills, competencies, proficiency levels, and evidence.
4. Credential layer: badges, certifications, and issued credentials.
5. Governance layer: lifecycle, review, quality gates, licensing, and trust scoring.
6. Publishing layer: website, search index, APIs, and JSON-LD export.
7. Agentic layer: curriculum, reviewer, mentor, and assessment agents.

## Principle

Every learning artifact is data, every relationship is explicit, every claim is reviewable, and every credential is backed by evidence.
