# Language-Neutral Core

OpenSaaSLabs Courses is a language-neutral, schema-first, governed learning graph.

## Canonical Layer

```text
schemas/
governance/
registry/
relations/
courses/
learning-paths/
competencies/
badges/
certifications/
evidence/
academy/
```

## Reference Implementation Layer

```text
tools/reference/python/
```

## Principle

The academy contract must survive replacement of any implementation language, framework, database, runtime, or hosting surface.

## Portable Implementations

Possible conforming implementations include:

- Python reference tools
- Go CLI tools
- Rust deterministic validator
- JavaScript static site generator
- JVM enterprise integration tooling

No implementation owns the model.
