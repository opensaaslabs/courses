# Community Governance

OpenSaaSLabs Academy follows an open, contributor-friendly governance model inspired by CNCF projects.

## Governance Layers

- Steering Committee
- Technical Oversight
- Special Interest Groups
- Maintainers
- Reviewers
- Contributors

## Principles

- Open decision-making
- Clear ownership
- Recorded decisions
- Conformance over implementation preference
- Security and quality before adoption
