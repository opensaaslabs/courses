# Tools

OpenSaaSLabs Courses is programming-language neutral.

The canonical platform contract is defined by schemas, governance, registries, relations, exports, and documented contracts. Tooling in any programming language is replaceable as long as it conforms to those contracts.

## Structure

```text
tools/
├── contracts/          # Language-neutral contracts
└── reference/          # Reference implementations
    └── python/         # Python reference implementation
```

## Rule

Reference tools are not canonical. They are one conforming implementation of the academy contracts.
