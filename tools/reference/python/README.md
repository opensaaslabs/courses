# Python Reference Implementation

This folder contains a reference implementation of the OpenSaaSLabs Academy contracts.

These scripts are provided for convenience and CI bootstrapping. They are not the canonical model.

## Scripts

- `validate.py`
- `graph_validate.py`
- `quality_score.py`
- `export.py`
- `site_generator.py`

## Contract Alignment

Each script should implement the corresponding language-neutral contract under `tools/contracts/`.
