# Economics Model

The repository should support multiple academy operating modes without changing the core graph model.

## Modes

- Public Academy
- Community Academy
- Partner Academy
- Enterprise Academy
- University Academy

## Offerings

- Free learning paths
- Community credentials
- Partner enablement
- Enterprise onboarding
- Workforce capability programs
- Certification programs

## Principle

The public graph remains open where possible. Enterprise-specific governance, evidence, and learner data must remain isolated.
