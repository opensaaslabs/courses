# Publishing Model

## States

```text
Draft -> Review -> Approved -> Published -> Deprecated -> Archived
```

## Draft
Content is editable and not canonical.

## Review
Content is checked for correctness, quality, completeness, and governance.

## Approved
Content has passed review and is ready for publication.

## Published
Content is visible in academy exports and public catalog surfaces.

## Deprecated
Content remains accessible but should not be used for new learning paths without migration guidance.

## Archived
Content is retained for auditability and provenance.

## Rule

Publication is allowed only when validation, quality, ownership, and licensing checks pass.
