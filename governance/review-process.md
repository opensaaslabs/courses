# Review Process

All publishable learning artifacts require review.

## Roles

- Author: creates or updates content.
- Reviewer: checks correctness, pedagogy, and maintainability.
- Approver: accepts the artifact for publication.
- Maintainer: owns lifecycle and updates.

## Minimum Review Checks

- Metadata is valid.
- Claims are sourced or clearly marked as original explanation.
- Outcomes are measurable.
- Labs and assessments map to outcomes.
- License is declared.
- Quality gate is passed before publication.
