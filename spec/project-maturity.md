# Project Maturity Model

OpenSaaSLabs Academy follows a CNCF-inspired maturity model.

## Sandbox
Experimental artifacts, early contracts, and incomplete implementations.

## Incubating
Stable enough for broader adoption. Requires governance, tests, conformance criteria, and active maintainers.

## Graduated
Production-ready, broadly usable, with clear governance, compatibility guarantees, security policy, conformance tests, and multiple implementations or adopters.

## Artifact Stability

- Experimental
- Alpha
- Beta
- Stable
- Deprecated
- Archived
